# Blade Proof System

Machine-checked kernel of the Blade formalism. Coq 8.18.0, stdlib only.

## Build

    coq_makefile -f _CoqProject -o Makefile
    make

or manually with `coqc -Q . Blade <file>` in _CoqProject order.
